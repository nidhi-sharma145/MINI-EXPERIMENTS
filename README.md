# MINI EXPERIMENTS

A collection of projects built while learning web development.

## Projects

#### Pet Adoption Page
A simple pet adoption webpage built using HTML and CSS.
Focus: Cards layout, images, and styling.

#### Camera Sidebar UI
A camera-themed sidebar interface created using HTML and CSS.
Focus: Navigation design and CSS positioning.

#### Spotify Homepage Clone
A clone of Spotify's homepage UI built with HTML and CSS.
Focus: Layout design, Flexbox, and responsive styling.

#### Todo List DOM
A dynamic todo list application built using JavaScript DOM manipulation.
Focus: Event handling, element creation, and deletion.

#### Simon Says Game
A memory-based color sequence game built with JavaScript.
Focus: Arrays, game logic, and DOM manipulation.

#### University Search
A university search application that fetches data from an external API using Axios.
Focus: API integration, Async/Await, and dynamic rendering.
